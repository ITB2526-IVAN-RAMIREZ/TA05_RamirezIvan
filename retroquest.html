Retroquest Gaming

E-Commerce platform specialized in nostalgia and collectibles.

About the Project

Retroquest Gaming was born from the need to centralize the classic video game market in a modern and secure platform. The goal was to create an environment that pays tribute to the golden era of 8- and 16-bit consoles without sacrificing current functionality.

The technical challenge consisted of developing an inventory system capable of handling specific conditions for used products, as well as providing a smooth search experience among thousands of retro titles.

Key Features

  • Secure payment gateway integrated with Stripe API.
  • Rating and reputation system for sellers and products.
  • Advanced filters by region (PAL/NTSC), console, and product condition.
  • Custom admin panel for managing orders and stock.

Tools

  • Frontend React & Tailwind CSS for a fast UI
  • Backend Node.js with Express for server-side logic
  • Database MongoDB for catalog scalability
  • Payments Stripe for secure transactions
  • Infrastructure Vercel & Docker for continuous deployment
