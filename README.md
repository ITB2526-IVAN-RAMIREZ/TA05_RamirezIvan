## Sobre el Proyecto

El objetivo de este proyecto fue construir una plataforma que sirviera como mi carta de presentación al mundo de tecnologico como CiberAdmin y Desarollador. Se centra en la **claridad visual**, la **jerarquía de la información** y la **interactividad fluida**. 

A diferencia de usar plantillas, he desarrollado cada componente (navegación, filtros de proyectos, tarjetas y formularios) utilizando estándares modernos de desarrollo web para garantizar un código limpio y mantenible.

---

## Stack Tecnológico

He utilizado un enfoque de "Vainilla" para demostrar un dominio profundo de las bases del desarrollo web:

* **HTML5:** Marcado semántico para optimizar la accesibilidad.
* **CSS:** * Uso intensivo de **Variables CSS (Custom Properties)** para una gestión de colores centralizada.
    * Layouts complejos mediante **Flexbox y CSS Grid**.
    * Animaciones y transiciones suaves para mejorar el *feedback* del usuario.
    * Estrategia **Mobile-First** para asegurar la responsividad en cualquier dispositivo.
* **JavaScript:** Lógica para la manipulación del DOM, sistema de filtrado dinámico de proyectos y gestión del formulario de contacto.

---

## Proyectos Destacados

El porfolio actúa como un gestor de proyectos dinámico, donde destacan:

* **RetroQuest Gaming Page:** Una landing page e-commerce diseñada con una estética "retro" pero funcional, enfocada en la venta de videojuegos clásicos.
* **Natura Google Sites:** Integración y diseño de una plataforma enfocada a la vida saludable y nutrición.
* **Filtro Dinámico:** El porfolio incluye una barra de filtrado funcional (Web Design, E-Commerce, Apps, SaaS, UI/UX) que permite al usuario navegar por mis habilidades de forma eficiente.

---

## Funcionalidades Técnicas

-   **Navegación Persistente:** Menú superior *Sticky* con efectos de transición.
-   **Dark Mode Nativo:** Diseño optimizado para reducir la fatiga visual con una paleta de colores `#121212` y acentos `#e74c3c`.
-   **Formulario de Contacto Real:** Integrado con **Formspree** para recibir mensajes directamente sin necesidad de un backend propio.
-   **Interacciones UI:** Efectos de *hover* en tarjetas de proyectos con elevación de profundidad (box-shadow) y escala.
